MirScore

An introduction to the Score Viewer.

Component overview

The Score Viewer is used for the graphical preview of scores.
There are three main modes for the application. In Pan mode the user can move inside the score by dragging the mouse. The Select mode allows the composer to set the playback position by clicking on a staff. During the MIDI Playback a graphical score element that represents the current position is updated.
The component supports pagination and zooming as well (during the playback they are adjusted to show the current portion of the score being executed).
